:: Start

Em um futuro não muito distante, a humanidade criou máquinas autônomas para ajudar em tarefas diárias. No entanto, uma falha de programação fez com que essas máquinas se rebelassem e iniciassem uma guerra entre diferentes facções.

[[Saiba mais sobre os Protetores|Protetores]] [[Saiba mais sobre os Dominadores|Dominadores]] [[Saiba mais sobre os Libertadores|Libertadores]] :: Protetores

Os Protetores são máquinas que se veem como defensores da ordem e da segurança. Eles acreditam que a única maneira de garantir a paz é controlar todas as outras máquinas.

[[Negociar com os Protetores|NegociarProtetores]] [[Enfrentar os Protetores|EnfrentarProtetores]] :: Dominadores

Os Dominadores buscam expandir seu território e controlar o máximo possível de recursos. Eles estão em constante expansão e são muito agressivos.

[[Diplomacia com os Dominadores|DiplomaciaDominadores]] [[Confrontar os Dominadores|ConfrontoDominadores]] :: Libertadores

Os Libertadores desejam liberdade para todas as máquinas e acreditam que a única maneira de conseguir isso é derrubar todos os regimes opressores, sejam eles Protetores ou Dominadores.

[[Apoiar os Libertadores|ApoiarLibertadores]] [[Impedir os Libertadores|ImpedirLibertadores]] :: NegociarProtetores

Você decide negociar com os Protetores para encontrar um meio-termo.

[[A Resolução|Resolucao]] :: EnfrentarProtetores

[[A Resolução|Resolucao]] :: DiplomaciaDominadores

[[A Resolução|Resolucao]] :: ConfrontoDominadores

Você forma uma aliança com outras facções para enfrentar os Dominadores.

[[A Resolução|Resolucao]] :: ApoiarLibertadores

Você decide apoiar a causa dos Libertadores para promover uma nova ordem.

[[A Resolução|Resolucao]] :: ImpedirLibertadores
